# Mini-CRM

Упрощённая CRM для управления клиентами. SPA на Vue 3.

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`.

## Сборка

```bash
npm run build
```

## Архитектурные решения

### Структура проекта

```
src/
├── api/           — мок API (localStorage + имитация задержки)
├── composables/   — переиспользуемые composables (useDebounce)
├── router/        — маршрутизация
├── stores/        — Pinia store для клиентов
├── types/         — TypeScript типы
└── views/         — страницы приложения
```

### Мок API

Вместо внешнего сервера используется localStorage — данные сохраняются между перезагрузками. При первом запуске генерируется 12 клиентов. Все операции возвращают Promise с небольшой задержкой для имитации сетевых запросов.

### State management

Pinia store (`useClientStore`) управляет состоянием клиентов, включая загрузку и ошибки. Store предоставляет стандартные CRUD-операции.

### Фильтрация и поиск

- Поисковая строка с **debounce** (300ms) для оптимизации
- Фильтры сохраняются в **query-параметрах URL** — при обновлении страницы состояние восстанавливается

### Дополнительные фичи

1. **Подтверждение удаления** — модальное окно перед удалением
2. **Toast-уведомления** — обратная связь при CRUD-операциях
3. **Debounce** — для поисковой строки
4. **Сохранение фильтров в URL** — через query-параметры
